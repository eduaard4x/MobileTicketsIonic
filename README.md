# MobileTicketsIonic

Aplicativo mobile desenvolvido com Ionic + Angular como atividade acadêmica.

## Sobre o projeto
Projeto criado para praticar desenvolvimento mobile híbrido utilizando Ionic com Angular e integração com Capacitor.

## ferramentas utilizadas
- Ionic Framework
- Angular
- Capacitor

## execução

npm install
ionic serve
